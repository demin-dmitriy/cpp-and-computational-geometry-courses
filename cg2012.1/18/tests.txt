Тестируется на тестах из задачи 01-convex_hull.

Они хороши по следующим причинам:

 - Есть простые тесты, которые можно визуализировать
 - Есть performance тесты, требующие много времени и памяти
 - Есть тесты на точность, в частности тесты, где точки почти совпадают
 - Есть тесты, где все точки находятся на одной прямой - единственный случай,
   в котором граф диаграммы вороного несвязен.
 - Есть тесты, где точки находятся на окружности - случай, где вершина имеет
   большую степень.

Не идеальны:
  - Нет теста, где много точек с общей x-координатой, и теста где много точек с
    y-координатой - проверка на ошибки реализации.
  - В некотрых тестах точки дублируются.
